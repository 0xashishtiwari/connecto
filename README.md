<h1 align="center" id="title">Connecto</h1>

<p align="center"><img src="https://socialify.git.ci/0xashishtiwari/connecto/image?font=JetBrains+Mono&amp;language=1&amp;name=1&amp;owner=1&amp;stargazers=1&amp;theme=Auto" alt="project-image"></p>

<p id="description">Connecto is a real-time video meeting application inspired by Zoom built using Next.js (TypeScript) on the frontend and a Node.js + Socket.IO backend. The project uses WebRTC for real-time audio and video communication and focuses on understanding how browsers establish live media connections through signaling ICE negotiation and session management. Connecto enables users to join room-based meetings exchange media streams in real time and demonstrates the core building blocks of modern video conferencing systems.</p>

```

```

<p>5. Start the signaling server:</p>

<p>7. Open a new terminal and navigate to the frontend folder:</p>
